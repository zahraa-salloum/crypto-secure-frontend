Supported Algorithms

RC4

Deprecated

Variable key-size stream cipher developed by Ron Rivest in 1987. Used in SSL/TLS and WEP. Educational implementation only.

  • ✓ Variable key size (40-2048 bits)
  • ✓ Fast encryption/decryption
  • ✗ Known biased keystream
  • ✗ Vulnerable to related-key attacks

A5/1

Deprecated

Stream cipher used in GSM mobile phone encryption. Based on three linear feedback shift registers (LFSRs).

  • ✓ 64-bit key size
  • ✓ Historical GSM encryption
  • ✗ Practical cryptanalysis exists
  • ✗ Not recommended for use
